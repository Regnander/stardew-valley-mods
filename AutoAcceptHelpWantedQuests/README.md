# Auto-Accept 'Help Wanted' Quests

***Note: This mod has only been tested on local multiplayer.***

Tired of making the daily trek to Pierre’s General Store just to check the bulletin board? This mod streamlines your gameplay by automatically adding daily "Help Wanted" quests directly to your journal the moment you wake up.

## Configuration

You can set this mod to ignore quests from villagers with max friendship, prioritizing quests that allows you to gain friendship points by editing the `config.json` file in the mod folder, or by using [Generic Mod Config Menu](https://www.nexusmods.com/stardewvalley/mods/5098) in-game.

## Language support

This mod is machine-translated into all in-game languages. If you have suggestions for better wording, please let me know in the Posts tab.

## Installation

1. Install the latest version of [SMAPI](https://www.nexusmods.com/stardewvalley/mods/2400).
2. Download this mod and unzip it into your `Stardew Valley/Mods` folder.
3. (Optional) Install [Generic Mod Config Menu](https://www.nexusmods.com/stardewvalley/mods/5098) to change settings directly in-game.
4. Run the game using SMAPI.
