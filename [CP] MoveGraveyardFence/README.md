# Move the Graveyard Fence

***Note: This mod has only been tested on local multiplayer.***

Tired of bumping into the fence when traveling past the graveyard on your way to the beach? This mod will move the entire fence one tile to the left.

## Installation

1. Install the latest version of [SMAPI](https://www.nexusmods.com/stardewvalley/mods/2400) and [Content Patcher](https://www.nexusmods.com/stardewvalley/mods/1915).
2. Download this mod and unzip it into your **Stardew Valley/Mods** folder.
3. Run the game using SMAPI.
